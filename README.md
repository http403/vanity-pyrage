# vanity-pyrage

Age vanity public key brute-forcer with questionable performance.

Inspired by [seaofmars/vanity-age]([seaofmars/vanity-age](https://github.com/seaofmars/vanity-age))

Powered by [woodruffw/pyrage](https://github.com/woodruffw/pyrage)

## Usage
```
usage: vanity-pyrage [-h] pattern

A vanity age key generation tool powered by pyrage

positional arguments:
  pattern     regex pattern

options:
  -h, --help  show this help message and exit
```

Output is identical to `age-keygen`
